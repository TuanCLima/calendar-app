# calendar-app

Angular Drap&amp;Drop calendar app exercise

## Requirements
  - Node.js
  - Angular CLI
 
## Installation instructions
  - cd `[project-folder]`
  - `npm install`
  - `ng serve` (page will load at http//localhost:4200)
